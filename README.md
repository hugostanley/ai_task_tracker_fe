# LLM-Powered Chat Task Tracker (Frontend)
This is a simple Frontend UI integrating the [backend server repo](https://github.com/hugostanley/ai_task_tracker) using ReactJS.

## Local Setup & Run
Install dependencies

```
pnpm install
```

Create a .env file and copy the following
```
VITE_API_BASE_URL=http://localhost:3000
```

Run the application
> Ensure that the backend server is alsop running.

```
pnpm dev
```
